Coding projects

Word Association Game

A game that fetches data from an API based on the chosen difficulty level. Built with React and NodeJS.

Orthonormalization Visualization

Python program that uses the Plotly library to create an interactive 3D scatter plot that visualizes the relationship between input vectors and their orthonormal counterparts.

Gradient Descent Visualization

Python program that uses the Plotly library to create a 3D surface plot of a two-dimensional function, f(x, y), and a 3D scatter plot of the optimization path of the gradient descent algorithm.

Book Club Web Application

Database-driven web application built with HTML, CSS, JS, PHP, SQL, and Bootstrap for Effat University's book club. It is designed to help the members of the club stay informed about upcoming events, view the list of books that have been read, and view the list of members.

Movie Rating System

Movie rating system implemented in Java using the Swing framework and a database developed using XAMPP. The system allows users to view a list of movies, add new movies, and rate existing movies. The movies and their ratings are stored in a database, which can be accessed and modified through the user interface.

School System

Program developed with C++ to manage the data for a school with students and courses using objects from the different classes, based on a UML class diagram.

Numerical Analysis

Various numerical methods implementation in MATLAB for solving mathematical equations, including: Bisection, Euler, Newton-Raphson, Runge-Kutta, Secant, and Taylor series methods.

Assembly List

Assembly program that finds the smallest element in a list that's given by the user. This is a simple project but it provides great understanding of the logic behind arrays and their representation in the memory.
